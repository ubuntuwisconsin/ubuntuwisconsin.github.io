Above is an IRC client. Follow the steps listed below if this is your first time using IRC and you plan on using it again. If this will be your only time using IRC, then just click Start. Do not continue at all if you are under 13 years of age.

  1. Make sure you have your email client up. If you don't have an email address, create a Gmail account here.
  2. Think of a new username and type it above. If you can't think of one, use your first initial and your last name.
  3. Press start.
  4. Type /msg nickserv register TYPEANEWPASSWORD you@example.com and press enter.
  5. Open the email that Freenode sent you, copy the verification line in, and press enter.

For now on, when you use this client, you will have to check the box next to I have a password and type in the password you specified. This will make sure that nobody can steal your username.

After this, just say hello! We don't bite! :)

Long Term IRC Usage

As much as we would like you to use this IRC client, in the long run, it isn't efficient. So below are several guides on how you can move forward depending on what you use on a daily basis.

Only an Ubuntu (or flavor) computer

Since this is assuming that you will have regular terminal access, this will show you how to get tmux and irssi set up on a shell account for an idling setup. This is useful for when people need to speak to you, as they can just leave a message and you can get back to them when you have the time. Just a note, this shell will detect if you try to make multiple accounts and it will shut down all shell services. So please, only make one shell account.

  1. Go to xshellz.com/signup, fill out the form, and click Create shell. For usage, you should put something like "IRC idling setup". Please use the same username as IRC. In the future, when I specify NICK, this is the nick(username) you use.
  2. The previous step should bring you to a page titled Shell Panel. Click on the manage button on the table.
  3. Scroll down, check I have a password, type in your password, and click start.
  4. Type !approval and press Enter. Wait until the bot reacts.
  5. Type !approve NICK and press Enter. Wait until the bot reacts.
  6. After you get started with this, every week you need to go to #xshellz and send this to the channel: !keep NICK
  7. Refresh the page and click the Manage button again.
  8. Open a terminal by pressing Ctrl + Alt + T.
  9. In the terminal, type ssh USERNAME@shell.xShellz.com and press Enter.
  10. Look at the panel for your password. Type that password in now.
  11. After you are logged in, type tmux and press Enter.
  12. Type irssi and press Enter.
  13. Type /SERVER ADD -auto -network Freenode irc.freenode.net 6667 IRCPASSWORD and press Enter.
  14. Make sure all other IRC clients are closed (refresh this page to log out of IRC).
  15. In irssi, type /exit and press Enter.
  16. Type irssi and press Enter.
  17. Now, you can type /CHANNEL ADD -auto #CHANNELNAME Freenode then /join #CHANNELNAME to join any channel you would like. Our channel is #ubuntu-us-wi, so do /CHANNEL ADD -auto #ubuntu-us-wi Freenode then /join #ubuntu-us-wi.
  18. Read the Irssi Startup HOWTO.
  19. To exit your IRC client for the day, to shut down your computer, do Ctrl + B and then D. You can then close the terminal window. To get back into the IRC client, do steps 8-10, and then type tmux attach.
