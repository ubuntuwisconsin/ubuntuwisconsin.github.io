Download Xubuntu 15.10

Name: Xubuntu

First release: 6.06 LTS

Initial release date: June 23, 2006

Desktop environment: Xfce

Description:

Xubuntu provides a beautiful, lightweight desktop for older computers and for newer computers that want some speed. It provides a lightweight experience as well as a beautiful desktop.

Xubuntu is not resource-intensive at all except for very old computers. Although it is very lightweight, Lubuntu is more lightweight.

System requirements:

Absolute minimum:

  • 384 MB of RAM
  • 1 GHz processor
  • 7 GB of hard drive space
  • VGA capable of 800x600 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media

Recommended:

  • 768 MB of RAM
  • 1.5 GHz processor
  • 20 GB of hard drive space
  • Any display capable of 1280x1024 or more
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
