What is Ubuntu?

Ubuntu itself

Ubuntu is an open source alternative to major platforms such as Windows or Mac OS X. ‘Ubuntu’ is an African word meaning ‘humanity to others’. It also means ‘I am what I am because of who we all are’, which aptly describes the open source and open community nature of the project. Ubuntu is the most popular Linux distribution there is, with millions of users worldwide. The project is backed by a company named Canonical, which sponsors the project and provides several useful services for it. How often do I have to update?

First and formost, you should frequently update your Ubuntu computer because every update fixes a lot of bugs, adds new features, and chances are, will make your computer run better.

Ubuntu has 2 release channels. Stable and Long Term Support(LTS).

A new LTS release is released every 2 years, and it is oriented towards stability. LTS releases are supported for 5 years after release. During the release cycle, it allows for much less time to add features, and focuses on bug fixing to make it as stable as possible.

A new stable release is released every 6 months and is oriented towards more features. During the release cycle, it allows for more time for developers to add new featues, and less time to fix bugs compared to an LTS release. Stable releases are supported for 9 months after release.
