Name: Lubuntu

First release: 10.04

Initial release date: April 29, 2010

Desktop environment: LXDE

Description:

Lubuntu provides a very lightweight desktop that is very customizable. It also looks great for a lightweight desktop.

You can use old computers as old as 2004, and new computers with Lubuntu. It runs very smoothly with almost all of them.

System requirements:

Absolute minimum:

  • 128 MB of RAM
  • 800 MHz processor
  • 5 GB of hard drive space
  • VGA capable of 800x600 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media

Recommended:

  • 1 GB of RAM
  • Dual core 1 GHz processor
  • 25 GB of hard drive space
  • Any display capable of 1280x1024 or more
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
