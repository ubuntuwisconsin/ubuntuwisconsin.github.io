Name: Mythbuntu

First release: 7.10

Initial release date: October 22, 2007

Desktop environment: Xfce

Description:

Mythbuntu is meant for media center machines. It provides the MythTV DVR suite built-in. You can have two systems with Mythbuntu, the frontend and the backend. The backend records all of the TV shows and videos, while the frontend plays them back.

Keep in mind that the frontend needs to be able to handle the video and the backend needs to be able to handle the recording.

System requirements:

Absolute minimum:

  • 2 GB of RAM
  • Dual core 2 GHz processor
  • 10GB + 1.5GB/hour for the backend, 2 GB for the frontend
  • VGA graphics card capable of 1024x768 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media
  • TV Tuner for the backend

Recommended:

  • 4 GB of RAM (frontend and backend)
  • Quad core 3 GHz processor (frontend and backend)
  • 1 TB of hard drive space for the backend, 15 GB for the frontend
  • Any display (with a graphics card) capable of 1280x1024 or more (frontend) and 45 fps as a good rule of thumb
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
  • TV Tuner for the backend
