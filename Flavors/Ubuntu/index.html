Name: Ubuntu

First release: 4.10

Initial release date: October 20, 2004

Desktop environment: Unity

Description:

Ubuntu is the default flavor. It was created in 2004 by Mark Shuttleworth, and it is based off of Debian, a popular Linux distribution. To learn more about the philosophy of the project, click here. Ubuntu used GNOME 2 as the desktop environment until 11.04, when the project transferred to Unity, which it still uses today. You can still use this Desktop Environment in Ubuntu MATE. Soon the transition will be made to Unity 8, which Ubuntu Phone users already use.

Ubuntu is fairly resource-intensive. Unity without any modifications uses 3 GB of RAM, which even on a 6-core machine with 8 GB of RAM, is taxing. If you have a very beefy computer, you can use Unity as-is, but if you are a regular consumer who bought a regular computer, then please use a more lightweight flavor, because as previously stated, it is resource-taxing. You can still use Unity, but you should then avoid taxing applications such as Google Chrome.

System requirements:

Absolute minimum:

  • 384 MB of RAM
  • 700 MHz processor (about Intel Celeron or better)
  • 5 GB of hard drive space
  • VGA capable of 1024x768 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media

Recommended:

  • 4 GB of RAM
  • Dual core 1.5 GHz processor
  • 25 GB of hard drive space
  • Any display capable of 1280x1024 or more
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
