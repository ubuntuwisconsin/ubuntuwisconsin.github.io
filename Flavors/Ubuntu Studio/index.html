Name: Ubuntu Studio

First release: 7.10

Initial release date: October 22, 2007

Desktop environment: Xfce

Description:

Ubuntu Studio is an Ubuntu flavor meant for "creative humans". It comes preinstalled with many creative tools.

384px Since Ubuntu Studio is meant for creative use, you will want hardware that tailors to what you do. For example, if you plan on doing audio editing, you might want an audio card.

System requirements:

Absolute minimum:

  • 1 GB of RAM
  • 1.5 GHz processor
  • 15 GB of hard drive space
  • VGA capable of 1024x768 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media

Recommended:

  • 4 GB of RAM
  • Quad core 1.5 GHz processor
  • 100 GB of hard drive space
  • Any display capable of 1280x1024 or more
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
