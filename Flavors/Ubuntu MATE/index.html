Name: Ubuntu MATE

First release: 15.04

Initial release date: April 23, 2015

Desktop environment: MATE

Description:

Ubuntu MATE brings the classic GNOME 2 interface back to life. MATE is the continuation of development for GNOME 2. It provides a lot of useful tweaks to make it a fast, modern interface.

Ubuntu MATE is about as resource-intensive as Xubuntu.

System requirements:

Absolute minimum:

  • 768 MB of RAM
  • 800 MHz processor
  • 9 GB of hard drive space
  • VGA capable of 1024x768 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media

Recommended:

  • 3 GB of RAM
  • Dual core 1.5 GHz processor
  • 25 GB of hard drive space
  • Any display capable of 1280x1024 or more
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
