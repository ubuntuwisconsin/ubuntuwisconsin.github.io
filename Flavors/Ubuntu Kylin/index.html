Name: Ubuntu Kylin

First release: 13.04

Initial release date: April 25, 2013

Desktop environment: Unity

Description:

Ubuntu Kylin provides an Ubuntu experience for Chinese users. It provides special software that makes the usability for Chinese users better. For this reason, if you do not speak Chinese, it is not recommended, but if you speak Chinese, this will work well for you.

Since Ubuntu Kylin is almost the same as Ubuntu, the recommendations are the same.

System requirements:

Absolute minimum:

  • 384 MB of RAM
  • 700 MHz processor (about Intel Celeron or better)
  • 5 GB of hard drive space
  • VGA capable of 1024x768 screen resolution
  • Either a CD/DVD drive or a USB port for the installer media

Recommended:

  • 4 GB of RAM
  • Dual core 1.5 GHz processor
  • 25 GB of hard drive space
  • Any display capable of 1280x1024 or more
  • Either a CD/DVD drive or a USB port for the installer media
  • Internet access
